Instructions for compiling shopping.py

1. Transfer all files and the shopping.txt file that will be read in for shopping.py
to the flip server. 
2. On flip call python3 shopping.py and a file named results.txt will contain the 
output of the maximmum total price of all goods that a family can carry out during
their shopping spree for each family member and the list of items each family member
should select.
